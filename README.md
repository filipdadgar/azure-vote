# Running
* git clone
* cd azure-vote
* docker-compose up (-d for background)
* docker-compose logs (to view the logs)
* docker ps -a (to view the running containers)
# Troubleshooting
> If you are welcomed by error 500, wait a few second for the db to warmup and hit refresh.
